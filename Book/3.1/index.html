CSS Selector Practice

(a) div + strong

This is a div. Inside it is a strong element.
Another div with bold/strong text.

This strong is NOT inside a div: strong outside div

(b) p + em (same rule)

This is a paragraph.

This paragraph contains emphasized text inside it.

This em element is outside of a paragraph.

(c) #Nevada + .shiny

This paragraph has id="Nevada".

This div has class="shiny".

This paragraph also has class="shiny".

This paragraph has id="California" (should NOT match Nevada).

(d) span.bigger

Normal text... This span has class bigger. ...back to normal.

This span has no class.

(e) span inside span

Outer span start... Inner span (this should match the descendant rule) ...outer span end.

This is a span NOT inside another span.

## Local Development

To run the project locally, clone the repository:

```bash
git clone https://github.com/ABUO023/Lab3-CSI-3540.git
```

Then open the downloaded folder (`Lab3-CSI-3540`) in your file manager or IDE.

To navigate through the lab exercises, open:

* `index.html` (in the root folder)

From there, you can click the exercise links and view any associated CSS files.
